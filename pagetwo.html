Question-1: What are the different ways to select an element in the DOM?

Answer: We can select an element in the DOM by selecting its ID, class name, tag name, or with CSS selectors such as querySelector and querySelectorAll.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer: InnerHTML returns the HTML content of an element,InnerText returns the text-content and textContent returns the text content of the chosena node and its child-nodes.

Question-3:What is event delegation in the DOM?

Answer: When a single parent element's event listener is used to handle events from its descendant child elements or nodes that is called event delegation.

Question-4: What is event bubbling in the DOM?

Answer: Event Bubbling is a concept where an element receives an event and that event bubbles up to its parent and ancestor elements in the DOM tree until it gets to the root element.

Question-5: How do you create, add, and remove elements using JavaScript?

Answer: We can use document.createElement() for creating an element, document.createTextNode() for adding an element and element.remove() for removing an element in JavaScript.
